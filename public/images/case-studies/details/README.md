# Case Study Detail Images

Paste your case study images into the corresponding project folder:

## Folder structure

```
details/
├── cardly/
│   ├── hero.jpg       — Main hero image (534×345 recommended)
│   ├── gallery-1.png  — Bottom-left gallery image
│   └── gallery-2.png  — Bottom-right gallery image
├── homify/
│   ├── hero.jpg
│   ├── gallery-1.png
│   └── gallery-2.png
└── scheweppes/
    ├── hero.jpg
    ├── gallery-1.png
    └── gallery-2.png
```

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
